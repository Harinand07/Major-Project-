# Blockchain Project

A blockchain implementation with a frontend web interface.

## Features

- Blockchain core functionality
- Web-based explorer interface
- Transaction management
- Wallet management
- Block viewer

## Project Structure

- `frontend/` - Frontend web application files
  - `index.html` - Main application page
  - `app.js` - Application logic
  - `explorer.js` - Blockchain explorer
  - `styles.css` - Styling
  - `blocks.html` - Block viewer
  - `transactions.html` - Transaction viewer
  - `wallet.html` - Wallet interface
  - `mempool.html` - Memory pool viewer
  - `login.html` - Login page

## Getting Started

1. Clone the repository
2. Navigate to the frontend directory
3. Open `index.html` in your browser
